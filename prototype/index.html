น้อง NOVA — น้องโซโซ่เวอร์ชัน Finovas

ยึดดีไซน์น้องโซโซ่ (เครือ SO/Finovas) เพื่อแบรนด์เดียวกัน: คาปิบาร่าตัวป่องกลม แก้มยุ้ย ตาโตหลังแว่น ยิ้มหวาน • เน็คไทฟ้า Finovas + สูทบาง ถือเครื่องคิดเลข — เป็นมิตร อบอุ่น น่าเชื่อถือ

LIFF Survey (มุมมองลูกค้า)

Back-office (Admin)

มุมมองหลังบ้าน: จัดโครงสร้างทีมบัญชี (หัวหน้า/ลูกทีม) + ผูกบริษัทกับผู้ดูแล แบบมีผลตามวันที่ (effective-dated) — คือข้อมูลที่แบบประเมินนักบัญชีดึงไปผูกอัตโนมัติ

เปิดพรีวิว (สำนักงาน) ▸

นักบัญชี (ผูกผู้ดูแลอัตโนมัติ)

รอบรายเดือน — ระบบผูกหัวหน้าทีม + ลูกทีมให้อัตโนมัติ แล้วขึ้นคำถามประเมินต่อคน (ลูกค้าไม่เลือกเอง)

เซล — ขายได้ (Won)

ส่งครั้งเดียวหลังปิดการขายได้ — ความพึงพอใจต่อเซล, เหตุผลที่ตัดสินใจซื้อ, ปลายเปิด

✓ กดดูได้จริงในต้นแบบ

  • Mascot 4 แบบ (Full / Icon / Profile / Loading)
  • LIFF Survey แบบ A เดินครบเส้นทาง (state machine ใน JS)
  • Conditional: คะแนน 1–2 เด้งหาสาเหตุ+ติดต่อกลับ, 4–5 ถามจุดเด่น
  • Multi-select "ยังไม่พบปัญหา" ตัดข้ออื่น
  • Progress bar + auto-save banner + จำลอง offline
  • Review ก่อนส่ง + Confirmation
  • Dashboard สลับมุมผู้บริหาร ↔ นักบัญชี

◐ จำลองไว้ / ยังไม่ทำจริง

  • ข้อมูลทั้งหมดเป็น mock — ไม่ต่อ backend/DB
  • LINE Login / LIFF SDK จริง (ในต้นแบบข้ามไป)
  • AI น้อง NOVA วิเคราะห์จริง (โชว์แต่ผลตัวอย่าง)
  • Auth/RBAC/RLS จริง — สลับมุมด้วยปุ่มเฉยๆ
  • กราฟ trend จริง (ใช้แท่ง/ตัวเลข mock)
  • การส่งลิงก์ invitation อัตโนมัติ (พรีวิว 4 ชนิดได้ที่ Back-office)
